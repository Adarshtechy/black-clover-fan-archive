## 🍀 Black Clover Fan Archive

A premium fan-made website dedicated to the world of **Black Clover**, featuring characters, kingdoms, magic attributes, story arcs, galleries, media, and more.

> **"Surpass your limits. Right here. Right now."**

## 📖 Overview

Black Clover Fan Archive is a fully responsive anime fan website inspired by Yūki Tabata's *Black Clover*. It provides an immersive experience for fans through modern UI design, smooth animations, and organized information about the Black Clover universe.

## ✨ Features

- 🎬 Cinematic Hero Sections
- 👥 Character Profiles
- ✨ Magic Attribute Encyclopedia
- 📚 Grimoire Information
- 🏰 Four Kingdoms Guide
- 📖 Story Overview & Timeline
- 🖼️ Interactive Gallery
- 🎵 Media & OST Section
- 📱 Fully Responsive Design
- ⚡ Smooth Animations & Transitions

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Adarshtechy/black-clover-fan-archive.git
```

Open the project

```bash
cd black-clover-fan-archive
```

Run

Simply open **entry.html** in your browser.
